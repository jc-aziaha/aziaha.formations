Le programme pédagogique

  • La durée

    Cette formation se déroule sur 5 jours (de 9h à 17h).

  • Le public

    Cette formation s'addresse aux :
    personnes désirant apprendre de nouvelles choses
    personnes désirant effectuer une reconversion professionnelle
    Webmasters débutants

  • Les pré-recquis

    Maîtrise de la langue française
    Savoir naviguer sur le web
    Etre motivé

  • Rappelons les objectifs généraux de la formation

    A la fin de ce cours, vous serez capable de :
    Comprendre le Web
    Ecrire du code HTML
    Réaliser une mise en page structurée en HTML
    Ecrire du code CSS
    Réaliser une mise en forme d'une page web avec du CSS
    Créer un site internet et adapter son affichage en fonction des différents écrans sur lesquels il sera affiché

  • Type de formation

    * Cette formation se déroule en distanciel synchrone.
    * C'est à dire que c'est une formation qui ne necessite pas de se rendre dans une salle de cours physiquement mais plutôt de manière virtuelle.
    * La présence de chaque apprenant est indispensable durant toute la durée de la formation.

  • Outils

    Un ordinateur en bon état et une bonne connexion internet
    Un navigateur mis à jour
    Un éditeur de code (Visual Studio Code)
    Le logiciel ZOOM sur votre ordinateur
    Cliquez pour télécharger ZOOM Cliquez pour télécharger Visual Studio Code

  • Le prix

    1500 € HT

  • Evaluation

    Un QCM à faire sera transmis à chaque apprenant vers la fin de la formation

  • Sanction

    Attestation de fin de stage mentionnant le résultat des acquis

  • Le programme de la formation
    Comprendre le fonctionnement du web
    - Connaître l'histoire du Web
    - Savoir identifier les principaux langages web : HTML, CSS, JS et PHP
    - Comprendre le fonctionnement client/serveur
    - Connaître l'évolution des sites web dans le temps
    Créer sa première page web
    - Comprendre le principe du HTML : balises et attributs
    - Connaître les deux familles de balise : avec et sans contenu
    - Insérer les balises essentielles : doctype, head, body, etc.
    - Insérer Les commentaires
    - Différencier entre les balises inline et block
    Intégrer du contenu dans une page en HTML

    - Insérer du contenu texte : titre ou paragraphe
    - Ajouter du contenu média : image, vidéo et audio
    - Créer des liens hypertextes
    - Embarquer un contenu externe : vidéo, cartographie, etc.
    - Créer des listes (ordonnées et non ordonnées)
    - Créer des tableaux
    - Créer des formulaires
    Atelier : Créer un menu de navigation et lui appliquer un style CSS
    Le CSS
    - Présentation et définition - Attacher une css à une page html - Les selecteurs (balise, id, class) - Les pseudo-selecteurs - Les propriétés et valeurs - Les commentaires Atelier : Attacher une feuille css à une page html
    Les propriétés
    - Les propriétés de texte (police, taille, alignement) - Les couleurs hexadecimale, rgb, rgba et nommées - Les propriétés d'arrière plan - Les marges intérieurs et extérieur (margin, padding) - Les bordures - Les dimensions - Les listes Atelier : Styliser une page existante
    Le positionnement de bloc
    - Positionnement absolu ou relatif - Bloc flottant et fixe - Les blocs flexibles - Les grilles Atelier : Positionner différents blocs sur une page
    Mettre en ligne son site web
    Atelier : Mettre en ligne le site web réalisé
  • Une pédagogie basée sur la pratique

    20% de théorie (avec support de cours foruni)
    80% de pratique (utilisation d'un éditeur de code et d'un navigateur)
